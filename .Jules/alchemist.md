## 2024-05-24 - Deterministic Byte-Stream State Machine for JSON Minification
**Learning:** For deterministic JSON minification in C++ without heavy AST dependencies, a byte-stream state machine that tracks strings and escape sequences is highly efficient and resilient. This approach features a two-pass strategy: scan first to calculate total required size, then a single reserve() and append pass. This strategy operates in linear time with minimal heap allocations.
**Action:** Apply this two-pass byte-stream processing paradigm to future text or format minifiers, where avoiding a full parse tree is necessary for deterministic and secure execution in memory-constrained or safety-critical contexts.
